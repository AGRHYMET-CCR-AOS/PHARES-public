
# PHARES Public Repository

Welcome to the public repository of the PHARES forecasting and early warning platform.

This repository contains public resources related to:
- PHARES documentation;
- user guides;
- training materials;
- practical exercises;
- workshop resources;
- operational support documentation;
- and the [PHARES Wiki](https://github.com/AGRHYMET-CCR-AOS/PHARES-public/wiki).

